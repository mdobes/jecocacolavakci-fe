# Je Coca-Cola v akci?
Front end for [jecocacolavakci.cz](https://jecocacolavakci.cz).

# Installation
Use the package manager npm to install dependencies.
```bash
npm install
```

# Usage
For prewiew use:
```bash
npm start
```

For build use:
```bash
npm run build
```
Then you can upload the dist directory onto your web server.
